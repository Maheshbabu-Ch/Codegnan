Colors
3 ways to add colors
  1. color name
    2. eg: blue, black etc...
  2. rgb format
    3. Here values are ranged form 0-255 eg: rgb(255,255,0)
  3. Hexacode
    4. values are ranged from [0-9 A-F] and they are in the form of #rrggbb eg: #00FA10
Background Properties
  1. background-color
    2. same like color values
  2. background-image
    3. url("path")
  3. background-attachment
    4. scroll fixed
  4. background-position
    5. top left, top right, center, bottom left, bottom right
  5. background-repeat
    6. no-repeat, repeat-x, repeat-y, repeat
  6. background-size
    7. width height numbers, cover, contain
The first 5 properties can be initialized using background(short hand)
